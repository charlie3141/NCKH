Translator + Native-accent Text-to-Speech (single-file)

Translate text between languages and have the browser speak the original or translated text using a voice whose language tag matches the chosen language (availability depends on the browser/OS).

If you do not choose a voice, the page will attempt to pick one matching the language tag (e.g., "en", "es", "fr"). Voice availability depends on browser + OS.
Notes:
  • This single-file page uses the public LibreTranslate instance for translations (https://libretranslate.de). Public instances may be rate-limited. For production use, host your own translator or use a paid translation API.
  • Text-to-speech uses the browser SpeechSynthesis voices; "native accent" depends on which voices are installed on your device and the browser's voice list.
